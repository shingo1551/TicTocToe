# from React to Stencil

[TicTacToe](https://reactjs.org/tutorial/tutorial.html)をReactとStencilで作る

- react: React版
- hooks: React(Hooks)版
- step1: React風
- step2: Stencil版
- step3: stencil-store (sessionStorageを使用しreloadしても再現)
- use-stencil: step3をWeb Componentとして利用
